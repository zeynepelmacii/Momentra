# ⏳ Momentra – Time Counter

Momentra is a beautifully designed iOS application that helps you track the moments that matter most.

Whether you're counting down to an upcoming event or celebrating how much time has passed since a special memory, Momentra keeps every milestone in one elegant place.

---

## ✨ Features

- ⏳ Countdown timers
- ⏱ Count-up timers
- ❤️ Custom emoji icons
- 📅 Date & time picker
- ⚡ Quick presets
- ✏️ Edit existing counters
- 📊 Progress tracking
- 🎨 Modern and minimal UI
- 📱 Native iOS experience

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/zeynepelmacii/Momentra.git
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npx expo start
```

Build for iOS

```bash
eas build --platform ios
```
